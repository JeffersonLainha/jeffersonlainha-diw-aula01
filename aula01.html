Exercício sobre tags HTML

Tags h1 a h6

As tags h1 até h6 servem para representar títulos de seções no código HTML, sendo que a tag h1 tem a fonte maior e vai diminuindo até a tag h6 que é a menor.

Tag p

A tag p serve para incluirmos um texto dentro de parágrafo.

Tag hr

A tag hr é usada para fazer uma linha horizontal na página.

Tag br

A tag br serve para quebra de linha na página.

Tag abbr

A tag abbr é utilizada para inserirmos abreviações na página, como por exemplo: CSS
